Sentiment Analysis

ArchAI uses sentiment analysis to determine if messages are positive, negative or neutral. Users with high negative message count are punished and asked to act positively.

Image Captioning

ArchAI can describe an image if it is sent one. It uses the vit-gpt2-image-captioning model to classify and caption images.

Conversational AI

ArchAI uses the blenderbot-400M-distill model to chat with users, answer questions and store user-specific facts to create personalized experiences.
